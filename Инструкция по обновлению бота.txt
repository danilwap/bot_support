Как вносить обновления в бота
1) Заходим на beget, файловый менеджер и загружаем туда обновлённого бота, желательно env на хостинге не трогаем, а если загружаете из архива, проверьте что нужные данные в нём.
2) Подключаемся к хостингу по ssh
3) Переходим в папку проекта cd help_bot

4) Получаем id контейнера, для этого вводим docker ps и копируем его
5) Останавливаем бота командой docker stop id_из_пункта_выше

6) очищаем старые данные
		docker image prune --all
		docker system prune -a

7) Собираем образ docker build -t help_bot .
8) Запускаем контейнер docker run -rm --restart always -d help_bot
